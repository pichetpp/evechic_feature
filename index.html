🏠
หน้าหลัก (Home)
Banner, สินค้าแนะนำ, โปรโมชั่น
  • ✓① Hero Banner Carousel — สไลด์โฆษณา/โปรโมชั่นด้านบน + จุด indicator
  • ✓② Featured Collection — รูปใหญ่พร้อมสินค้าแนะนำ 3 ชิ้นวางซ้อน
  • ✓③ Banner โปรโมชั่น — สไลด์แบนเนอร์ + ปุ่ม "ช้อปเลย"
  • ✓④ สินค้าทั้งหมด — Grid 3 คอลัมน์: รูป, ชื่อ, ราคา, % ลด, ปุ่มหัวใจ
  • ✓Pull-to-refresh รีเฟรชข้อมูล
  • ✓Infinite Scroll โหลดสินค้าเพิ่มอัตโนมัติ
  • ✓ไอคอน Notification แจ้งเตือน (จุดแดงเมื่อมีข้อความใหม่)
👗
รายละเอียดสินค้า
ดูสินคะเต็มรูปแบบ + เลือกตัวเลือก + เพิ่มลงตะกร้า
  • ✓สไลด์รูปภาพสินค้าหลายรูป + Thumbnail + Indicator
  • ✓ชื่อ, แบรนด์, ราคา, ราคาตัด (% ลด)
  • ✓เลือกไซส์ — S, M, L, XL ฯลฯ
  • ✓เลือกสี — จุดสี hex code
  • ✓เลือกจำนวน — ปุ่ม + / -
  • ✓⭐ คะแนนเฉลี่ย + กราฟแท่งสัดส่วน + รีวิวผู้ซื้อจริง
  • ✓ปุ่ม "เพิ่มไปยังรถเข็น" (ต้องล็อกอินก่อน)
  • ✓ปุ่มหัวใจเพิ่มรายการโปรด
  • ✓แนะนำ "สินค้าที่คุณอาจสนใจ" เลื่อนแนวนอน

  • ✓แสดงรายการ: รูป, ชื่อ, ไซส์, สี, ราคา
  • ✓ปรับจำนวน + / -
  • ✓ลบสินค้า (ยืนยันก่อนลบ)

  • ✓Badge จำนวนบนไอคอนตะกร้า
  • ✓Zustand Cart Store ซิงค์กับ API
💳
ขั้นตอนสั่งซื้อ (Checkout)
3 ขั้นตอน: จัดส่ง → ชำระเงิน → ยืนยัน

📍 ขั้นที่ 1 — ที่อยู่จัดส่ง

  • ✓ชื่อ, ที่อยู่, อำเภอ, จังหวัด, รหัสไปรษณีย์, เบอร์โทร

🚚 เลือกวิธีจัดส่ง

  • ✓จัดส่งฟรี (3-7 วัน) / ธรรมดา 23฿ (4-6 วัน) / เร็ว 40฿ (2-3 วัน)

💳 ขั้นที่ 2 — ชำระเงิน

  • ✓เงินสด (COD) / บัตรเครดิต / พร้อมเพย์-โอนเงิน
  • ✓สรุปรายการสินค้า + ราคา + ค่าจัดส่ง + ยอดรวม

✅ ขั้นที่ 3 — สั่งซื้อสำเร็จ

  • ✓หน้าสำเร็จ + ข้อความขอบคุณ + ปุ่มดูออเดอร์ / กลับไปซื้อต่อ
  • !⏰ รอชำระเงินไม่เกิน 15 นาที — หากไม่ชำระภายใน 15 นาที ระบบจะยกเลิกรายการอัตโนมัติ + คืนสต็อกสินค้า + แจ้งเตือนลูกค้า

  • ✓แสดง: หมายเลข, วันที่, สินค้า, ยอดรวม, สถานะ
  • ✓Pull-to-refresh

📊 สถานะคำสั่งซื้อ (8 สถานะ)

🟡 รอชำระเงิน 🟡 ยืนยันแล้ว 🟡 กำลังเตรียม 🔵 จัดส่งแล้ว 🟢 จัดส่งสำเร็จ 🔴 ยกเลิก 🔴 คืนเงิน 🟠 มีปัญหารอแก้ไข

🚚 ติดตามพัสดุ (Tracking)

  • ✓ข้อมูลผู้ขนส่ง + หมายเลขติดตาม + ปุ่มคัดลอก
  • ✓"🎯 คาดว่าจะได้รับภายใน"
❤️
รายการโปรด (Wishlist)

  • ✓แจ้งเตือนที่ยังไม่อ่าน → จุดสีชมพู + พื้นหลังต่าง
  • ✓ปุ่ม "อ่านทั้งหมด" / แตะเพื่ออ่าน
  • ✓ประเภท: ยืนยันออเดอร์ • จัดส่ง • ชำระเงิน • โปรโมชั่น • ระบบ
📍

  • ✓แบรนด์, SKU, จำนวนในสต็อก
